# Presets

Presets are optionally packages that allows you to extend **siimple** by reusing themes and styles. 

```js title=siimple.config.js
import theme from "my-theme-preset";

export default {
    ...theme,
    // ...other config
    styles: {
        ...theme.styles,
        // ...custom styles
    },
};
```

## Official presets

We maintain the following presets, that can be optionally installed and used as a starting point or for creating new presets or extending your own themes:

- [@siimple/preset-reboot](/presets/reboot): adds CSS reset styles.
- [@siimple/preset-markup](/presets/markup): adds styles for markup elements.
- [@siimple/preset-utilities](/presets/utilities): adds essential utilities.
- [@siimple/preset-icons](/presets/icons): adds pure CSS icons.
