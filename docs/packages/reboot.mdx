# Reboot preset

This preset provides a modern and generic CSS reset that can be used to reduce inconsistencies between different browsers.

## Adding reboot preset

This preset can be installed using **npm**:

```bash
$ npm install --save @siimple/preset-reboot
```

In your configuration file, include the preset styles:

```js title=siimple.config.js
import reboot from "@siimple/preset-reboot";

export default {
    // ...other configuration
    styles: {
        ...reboot.styles,
        // ...other styles
    },
};
```
