# Icons Preset

We provide a collection of hand-crafted and pure CSS icons that can be used as a preset for the **siimple** toolkit. After enabling the preset, you can use the classname pattern `icon-<icon-name>` for including the icons:

```html
<i class="icon-star"></i>
```

## Adding icons preset

This preset can be installed using **npm**:

```bash
$ npm install --save @siimple/preset-icons
```

In your configuration file, include the preset styles:

```js title=siimple.config.js
import icons from "@siimple/preset-icons";

export default {
    // ...other configuration
    styles: {
        ...icons.styles,
        // ...other styles
    },
};
```

## Using icons

Create a `<i>` tag and add the `icon-<icon-name>` classname, where `<icon-name>` is the name of the desired icon.

```html 
<!-- Add a left arrow icon -->
<i class="icon-arrow-left"></i>

<!-- Add a star icon -->
<i class="icon-star"></i>
```

You can style your icon using `font-size` and `color` style properties:

```html 
<!-- Using style attribute -->
<i class="icon-star" style="color:#ffea66;font-size:20px;"></i>

<!-- Using helpers -->
<i class="icon-star has-color-red-500 has-size-xl"></i>
```
