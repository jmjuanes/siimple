# Reboot

This core plugin provides a modern and generic CSS reset that must be used to reduce inconsistencies between different browsers.

### Disable reboot styles

The reboot styles are enabled by default, but can be disabled setting to `false` the `useRebootStyles` in your configuration:

```js title=siimple.config.js
export default {
    useRebootStyles: false, // Skip reboot
};
```
