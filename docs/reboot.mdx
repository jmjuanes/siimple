# Reboot

This core plugin provides a modern and generic CSS reset that must be used to reduce inconsistencies between different browsers.

### Enable reboot plugin

The reboot plugin are enabled by default, but can be disabled setting to `false` the `reboot` field in the `corePlugins` section of the configuration.

```js title=siimple.config.js
export default {
    corePlugins: {
        reboot: false, // Skip reboot
    },
};
```
