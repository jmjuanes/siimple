# Tab

A simple navigation component. Style your tabs using the `tab` class, and group all of them using our additional class `tabs`. 

We also provide an `active` variant to mark the current tab.

```html live=true
<div class="tabs has-mb-0">
    <a class="tab">Link 1</a>
    <a class="tab is-active">Link 2</a>
    <a class="tab">Link 3</a>
</div>
```

### Variants

Variants for this component can be defined in the `navigation.tab` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "navigation.tab": (
            "default": (
                "background-color": "transparent",
                "&:hover": (
                    "color": "primary",
                ),
            ),
            "active": (
                "background-color": "white",
                "color": "primary",
            ),
        ),
    ),
);
```
