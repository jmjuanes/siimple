# Column

You can add **columns** to your grid row using the class `siimple-column`, and wrapp all columns in a `siimple-columns`. By default, you do not have to specify the size of the column: **each column will have by default an equal width**, proportional to the number of columns you have added.

```html live=true
<div class="siimple-columns has-text-center">
    <div class="siimple-column">
        <div class="has-radius has-bg-primary has-text-white has-p-2">Column 1</div>
    </div>
    <div class="siimple-column">
        <div class="has-radius has-bg-primary has-text-white has-p-2">Column 2</div>
    </div>
    <div class="siimple-column">
        <div class="has-radius has-bg-primary has-text-white has-p-2">Column 3</div>
    </div>
</div>
```

To customize the width of each column, we provide the following variants:

| Variant | Column size |
|---------|-------------|
| `one-fifth` | `width: 20%` |
| `one-quarter` | `width: 25%` |
| `one-third` | `width: 33.33%` |
| `two-fifths` | `width: 40%` |
| `half` | `width: 50%` |
| `three-fifths` | `width: 60%` |
| `two-thirds` | `width: 66.66%` |
| `three-quarters` | `width: 75%` |
| `four-fifths` | `width: 80%` |
| `full` | `width: 100%` |


```html live=true
<div class="siimple-columns has-text-center">
    <div class="siimple-column is-two-thirds">
        <div class="has-radius has-bg-primary has-text-white has-p-2">2/3</div>
    </div>
    <div class="siimple-column is-one-third">
        <div class="has-radius has-bg-primary has-text-white has-p-2">1/3</div>
    </div>
</div>
```

You can also apply these variants in different breakpoints to customize the column behavior across different devices and screen sizes.

You can combine different column classes to specify how the grid will be displayed on different devices. For example, if you create a grid with two columns and add the class `is-half` and `is-full-mobile`, your grid will have two columns on tablet screens and up, and one column on mobile devices.

```html
<div class="siimple-columns has-text-center">
    <div class="siimple-column is-half is-full-mobile">Column</div>
    <div class="siimple-column is-half is-full-mobile">Column</div>
</div>
```

You can hide a column at a specific breakpoint using the responsive `hidden` variant.

```html
<div class="siimple-columns">
    <div class="siimple-column is-one-third is-hidden-mobile">
        Hidden on mobile
    </div>
    <div class="siimple-column is-two-thirds is-full-mobile">
        Always visible
    </div>
</div>
```


### Variants

Variants for this component can be defined in the `column` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "column": (
            "default": (
                "padding": "0.75rem",
            ),
        ),
    ),
);
```
