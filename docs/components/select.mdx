# Select

You can style your `<select>` elements adding a `select` class.

```html live=true
<select class="select">
    <option selected>Option 1</option>
    <option>Option 2</option>
    <option>Option 3</option>
</select>
```


### Variants

Variants for this component can be defined in the `form.select` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "form.select": (
            "default": (
                "background-color": "fill",
            ),
        ),
    ),
);
```
