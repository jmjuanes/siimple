# Card

A **card** is a flexible container component that holds content and actions about a single subject or topic. By default, cards have no fixed width. A card can be created adding a `card` class to a `<div>` tag.


```html live=true
<div class="card has-mx-auto has-maxw-96">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
</div>
```

### Variants

Variants for this component can be defined in the `card` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "card": (
            "default": (
                "background-color": "white",
                "box-shadow": "0 0 0.5rem rgba(0, 0, 0, 0.125)",
                "color": "inherit",
            ),
        ),
    ),
);
```
