# Textarea

Add the `siimple-textarea` class to your `<textarea>` element. 

```html live=true
<textarea class="siimple-textarea" rows="5">Textarea content</textarea>
```

### Variants

Variants for this component can be defined in the `form.textarea` section of the theme configuration:

```scss
siimple.theme((
    "form.textarea": (
        "default": (
            "background-color": "fill",
        ),
    ),
));
```
