# Textarea

Add the `textarea` class to your `<textarea>` element. 

```html live=true
<textarea class="textarea" rows="5">Textarea content</textarea>
```

### Variants

Variants for this component can be defined in the `form.textarea` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "form.textarea": (
            "default": (
                "background-color": "fill",
            ),
        ),
    ),
);
```
