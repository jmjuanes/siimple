# Divider

The **divider** element helps you to separate content in your page. Add `siimple-divider` class to a `<div>` or a `<span>` tag to style it as a horizontal rule.

```html live=true
<p>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
    tempor incididunt ut labore et dolore magna aliqua.
</p>
<div class="siimple-divider"></div>
<p class="has-mb-0">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
    tempor incididunt ut labore et dolore magna aliqua.
</p>
```

### Variants

Variants for this component can be defined in the `divider` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "divider": (
            "default": (
                "background-color": "fill",
            ),
        ),
    ),
);
```
