# Slider

> This component is part of the **experimental** components module.

Use the `siimple-slider` class to style your range input elements:

```html live=true
<input type="range" class="siimple-slider" value="50" />
```


### Variants

Variants for this component can be defined in the `form.slider` section of the variants configuration:

```scss
@use "siimple" with (
    $variants: (
        "form.slider": (
            "default": (
                "background-color": "fill",
                "color": "primary",
            ),
        ),
    ),
);
```
