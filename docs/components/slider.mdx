# Slider

Use the `slider` class to style your range input elements:

```html live=true
<input type="range" class="slider" value="50" />
```


### Variants

Variants for this component can be defined in the `form.slider` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "form.slider": (
            "default": (
                "background-color": "fill",
                "color": "primary",
            ),
        ),
    ),
);
```
