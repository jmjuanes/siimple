# Title

Use `title` to style your headings. By default we provide six different variants to customize the title text size:

```html live=true
<div class="title is-1">Title 1</div>
<div class="title is-2">Title 2</div>
<div class="title is-3">Title 3</div>
<div class="title is-4">Title 4</div>
<div class="title is-5">Title 5</div>
<div class="title is-6">Title 6</div>
```

### Variants

Variants for this component can be defined in the `text.headings` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "text.headings": (
            "default": (
                "color": "heading",
                "font-weight": 600,
            ),
        ),
    ),
);
```
