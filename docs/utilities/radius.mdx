# Radius

The **radius** plugin provides a collection of helpers to control the border radius of any element. Included classnames are:

| Class | Property |
|-------|----------|
| `has-radius-sm`   | `border-radius: 0.25rem;` |
| `has-radius`      | `border-radius: 0.5rem;` |
| `has-radius-lg`   | `border-radius: 0.75rem;` |
| `has-radius-xl`   | `border-radius: 1rem;` |
| `has-radius-full` | `border-radius: 9999px;` |
| `has-radius-none` | `border-radius: 0px;` |

### Usage

You can use these classes to apply different border radius to your elements:

```html live=true centered=true bg=blue-200 color=white
<div class="has-p-6 has-m-2 has-bg-blue-500 has-radius-sm">
    <strong>.has-radius-sm</strong>
</div>
<div class="has-p-6 has-m-2 has-bg-blue-500 has-radius">
    <strong>.has-radius</strong>
</div>
<div class="has-p-6 has-m-2 has-bg-blue-500 has-radius-lg">
    <strong>.has-radius-lg</strong>
</div>
<div class="has-p-6 has-m-2 has-bg-blue-500 has-radius-xl">
    <strong>.has-radius-xl</strong>
</div>
```

You can use the `has-radius-full` class to create pills or circles:

```html live=true centered=true bg=blue-200 color=white
<div class="has-px-6 has-py-3 has-m-2 has-bg-blue-500 has-radius-full">
    <strong>Pill element</strong>
</div>
<div class="has-s-24 has-m-2 has-bg-blue-500 has-flex has-items-center has-justify-center has-radius-full">
    <strong>Circle</strong>
</div>
```

You can remove the border radius using `has-radius-none`: 

```html live=true centered=true bg=blue-200 color=white
<div class="has-p-6 has-m-2 has-bg-blue-500 has-radius-none">
    <strong>.has-radius-none</strong>
</div>
```
