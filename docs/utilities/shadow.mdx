# Shadow

Use the following shadow utilities to apply a box shadow effect to the desired element:

| Class | Property |
|-------|----------|
| `has-shadow-sm`  | `box-shadow: 0 .25rem 1rem -.125rem rgba(54,63,79,.15),0 0 0 1px rgba(54,63,79,.02);` |
| `has-shadow` | `box-shadow: 0 .5rem 1rem -.25rem rgba(54,63,79,.2),0 0 0 1px rgba(54,63,79,.02);` |
| `has-shadow-lg`  | `box-shadow: 0 1rem 1rem -.5rem rgba(54,63,79,.25),0 0 0 1px rgba(54,63,79,.02);` |
| `has-shadow-none`   | `box-shadow: none;` |

Control the shadow applied to any element using the shadow helpers:

```html live=true centered=true bg=blue-200
<div class="has-shadow-sm has-m-2 has-p-6 has-radius has-bg-white">
    .has-shadow-small
</div>
<div class="has-shadow has-m-2 has-p-6 has-radius has-bg-white">
    .has-shadow-normal
</div>
<div class="has-shadow-lg has-m-2 has-p-6 has-radius has-bg-white">
    .has-shadow-large
</div>
```

Remove the shadow of any element using `has-shadow-none`:

```html live=true centered=true bg=blue-200
<div class="has-shadow-none has-m-2 has-p-6 has-radius has-bg-white">
    .has-shadow-none
</div>
```
