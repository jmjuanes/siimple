# Color

A collection of utilities classes to customize the text and background color of any element. 

### Background color

Use `has-bg-{color}` to set the background color of any element. 

```html live=true bg=blue-200 align=center
<div class="button has-bg-royal-600 has-text-white">
    <strong>Button</strong>
</div>
```

#### Background color on hover

You can change the background color of any element on hover adding the `hover:` prefix to the background color utility.

```html live=true bg=blue-200 color=blue-700 align=center
<div class="button hover:has-bg-green-600 has-bg-aqua-600 has-text-white">
    <strong>Hover me</strong>
</div>
```

### Text color

Use `has-text-{color}` to apply a different color to the text. 

```html live=true bg=blue-200 align=center
<strong class="has-text-royal-600">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

#### Text color on hover

You can change the text color of any element on hover adding the `hover:` prefix to the text color utility.

```html live=true bg=blue-200 align=center
<strong class="has-text-blue-700 hover:has-text-red-400">Hover me</strong>
```
