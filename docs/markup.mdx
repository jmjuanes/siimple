# Markup

Use this plugin to style base markup elements, like headings, paragraphs, links... 

### Enable markup plugin

Markup styles are enabled by default, but can be disabled setting to `false` the `markup` field in the `corePlugins` section of the configuration.

```js title=siimple.config.js
export default {
    corePlugins: {
        markup: false, // Skip markup styles
    },
};
```

### Markup styles

#### Headings

This plugin will style your heading elements, from `<h1>` through `<h6>`. For styling non heading HTML elements, you can use the [title](/elements/title) element instead.

```html live=true
<h1>h1. Siimple heading</h1>
<h2>h2. Siimple heading</h2>
<h3>h3. Siimple heading</h3>
<h4>h4. Siimple heading</h4>
<h5>h5. Siimple heading</h5>
<h6>h6. Siimple heading</h6>
```

Headings will use the default variant of `text.heading` defined in the `variants` section of your configuration:

```js title=siimple.config.js
export default {
    variants: {
        "text.heading": {
            default: {
                color: "heading",
                fontFamily: "heading",
                fontWeight: "heading",
                lineHeight: "heading",
            },
        },
    },
};
```

#### Paragraph

This plugin will style your `<p>` elements. For style non paragraph HTML elements like `<div>` elements, you can use the [paragraph](/elements/paragraph) element instead.

```html live=true
<p class="has-mb-0">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua.
</p>
```

This plugin will use the default variant of `text.paragraph` defined in the `variants` section of your configuration:

```js title=siimple.config.js
export default {
    variants: {
        "text.paragraph": {
            default: {
                fontWeight: "body",
                lineHeight: "body",
            },
        },
    },
};
```

#### Blockquotes

This plugin will style your `<blockquote>` elements:

```html live=true
<blockquote>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua.
</blockquote>
```

#### Inline text elements

This plugin will style your inline text elements, like `<small>` or `<code>` elements.

```html live=true
<p>You can style your <a href="">link</a> elements.</p>
<p>You can style your <small>small text</small> elements.</p>
<p class="has-mb-0">You can style your <code>inline code</code> elements.</p>
```

#### Horizontal rule

This plugin will style your `<hr>` elements. For styling non `<hr>` elements as an horizontal rule, you can use the [divider](/elements/divider) element instead.

```html live=true
<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod...</p>
<hr>
<p class="has-mb-0">Lorem ipsum dolor sit amet, consectetur adipiscing elit...</p>
```

The horizontal rule will use the default variant of `divider` defined in the `variants` section of your configuration.

```js title=siimple.config.js
export default {
    variants: {
        divider: {
            default: {
                backgroundColor: "fill",
            },
        },
    },
};
```
