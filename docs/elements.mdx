# Elements

We provide a collection of built-in UI elements that you can customize and reuse across your projects, that includes buttons, alerts, navigation links, and more!

### Enable elements plugin

Elements are enabled by default, but can be disabled setting to `false` the `elements` field in the `corePlugins` section of the configuration.

```js title=siimple.config.js
export default {
    corePlugins: {
        elements: false, // Skip all elements
    },
};
```

Also, you can disable only individual elements just setting to `false` the element name in the `corePlugins` section of the configuration:

```js title=siimple.config.js
export default {
    corePlugins: {
        alert: false,
        button: false,
        card: false,
    },
};
```

### Customize elements

Variants can be used to customize and change the look and feel of elements. Each element will look for a specific variant defined in the `variants` section of your configuration.

```js title=siimple.config.js
export default {
    variants: {
        button: {
            default: {
                backgroundColor: "primary",
                color: "white",
            },
            secondary: {
                backgroundColor: "secondary",
            },
        },
    },
};
```
