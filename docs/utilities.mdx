# Utilities

The **utilities** plugin provides a collection of **small functional CSS classes** that can be used for creating or customizing components. The main aim of this plugin is to provide a collection of reusable classes to replace the conventional style of writting CSS classes.

### Enable utilities plugin

The utilities plugin are enabled by default, but can be disabled setting to `false` the `utilities` field in the `corePlugins` section of the configuration.

```js title=siimple.config.js
export default {
    corePlugins: {
        utilities: false, // Skip utilities
    },
};
```

Also, you can disable only individual utilities just setting to `false` the utility name in the `corePlugins` section of the configuration:

```js title=siimple.config.js
export default {
    corePlugins: {
        backgroundColor: false,
        display: false,
        textColor: false,
    },
};
```
