# Spacing

Customize the **padding** and the **margin** properties of anly element using the following default sizes:

| Size     | Value     |
|----------|-----------|
| `auto`   | `auto`    |
| `0`      | `0px`     |
| `1`      | `0.25rem` |
| `2`      | `0.5rem`  |
| `3`      | `0.75rem` |
| `4`      | `1rem`    |
| `6`      | `1.5rem`  |
| `8`      | `2rem`    |
| `12`     | `3rem`    |
| `16`     | `4rem`    |
| `24`     | `6rem`    |
| `32`     | `8rem`    |
| `48`     | `12rem`   |
| `64`     | `16rem`   |
| `96`     | `24rem`   |
| `128`    | `32rem`   |
| `256`    | `64rem`   |
| `half`   | `50%`     |
| `full`   | `100%`    |

> Any other sizes defined in the `spacing` object of your theme will generate additional spacing helpers.

### Uniform padding or margin

Use `has-p-{size}` to set the **padding** of all sides of the element, or `has-m-{size}` to set the **margin** of all sizes of the element.

```html live=true centered=true color=white
<div class="has-p-6 has-bg-blue-500 has-radius">
    <strong>has-p-6</strong>
</div>
```

### Directional padding or margin

Use `has-p{direction}-{size}` or `has-m{direction}-{size}` to set the **padding or margin of one side**, **on the x-axis** or **the y-axis** of the element. The following sides are available:

| Direction | Description |
|-----------|-------------|
| **t** | To set the margin or padding **top**. |
| **b** | To set the margin or padding **bottom**. |
| **l** | To set the margin or padding **left**. |
| **r** | To set the margin or padding **right**. |
| **x** | To set the margin or padding **left and right**. |
| **y** | To set the margin or padding **top and bottom**. |

```html live=true centered=true color=white padding=12
<div class="has-p-4 has-pt-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pt-12</strong>
</div>
<div class="has-p-4 has-pr-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pr-12</strong>
</div>
<div class="has-p-4 has-pb-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pb-12</strong>
</div>
<div class="has-p-4 has-pl-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pl-12</strong>
</div>
```

### Centering elements

We provide an additional `has-mx-auto` classname to set the margin left and right to `auto`. This will center the block with a fixed width.

```html live=true color=white
<div class="has-mx-auto has-p-3 has-w-32 has-bg-blue-500 has-radius">
    <strong>Centered</strong>
</div>
```

### Responsive spacing

You can control the spacing of an element at a specific breakpoint adding a `-{breakpoint}` suffix to the padding or margin class.

```html
<div class="has-p-3 has-p-0-mobile">
    . . .
</div>
```
