# Positioning

The **positioning** plugin provides a collection of helpers to control the position of any element.

### Enable this plugin

This plugin is included as part of the `plugin-helpers` collection. You can manually include this plugin calling `plugin-positioning-helpers()` in the `$plugins` section of the configuration:

```scss
@use "siimple" as siimple;

@include siimple.configure(
    $plugins: (
        siimple.plugin-positioning-helpers(),
    ),
);
```


### Position 

Use the following classes to change the position of an element:

| Class | Property | 
|-------|----------|
| `has-position-initial`  | `position: initial;` |
| `has-position-relative` | `position: relative;` |
| `has-position-absolute` | `position: absolute;` |
| `has-position-sticky`   | `position: sticky;` |
| `has-position-fixed`    | `position: fixed;` |


### Float

Use the following classes for positioning and formatting content, for example to wrap text around images.

| Class | Property | 
|-------|----------|
| `has-float-left`  | `float: left;` |
| `has-float-right` | `float: right;` |
| `has-float-none`  | `float: none;` |

### Placement

Use the following classes to set the placement of an element:

| Class | Property |
|-------|----------|
| `has-bottom-auto` | `bottom: auto;` |
| `has-bottom-0`    | `bottom: 0px;`  |
| `has-bottom-half` | `bottom: 50%;`  |
| `has-bottom-full` | `bottom: 100%;` |
| `has-left-auto`   | `left: auto;` |
| `has-left-0`      | `left: 0px;`  |
| `has-left-half`   | `left: 50%;`  |
| `has-left-full`   | `left: 100%;` |
| `has-right-auto`  | `right: auto;` |
| `has-right-0`     | `right: 0px;`  |
| `has-right-half`  | `right: 50%;`  |
| `has-right-full`  | `right: 100%;` |
| `has-top-auto`    | `top: auto;` |
| `has-top-0`       | `top: 0px;`  |
| `has-top-half`    | `top: 50%;`  |
| `has-top-full`    | `top: 100%;` |



### Responsive position

Control the position of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the position or placement class helper.

```html
<div class="mobile:has-position-fixed mobile:has-top-0">
    . . .
</div>
```
