# Typography

Customize the text properties using the following typography helpers.

### Font family

For each size defined in the `fonts` field of your theme there will be a helper with the pattern `has-font-{name}` for applying the specified font. The following fonts helpers will be always available:

| Class | Value |
|-------|----------|
| `has-font-body` | The font defined in the **body** field. By default it will apply `font-family: -apple-system, system-ui, BlinkMacSystemFont, "Segoe UI", Roboto, Ubuntu, "Noto Sans", "Helvetica Neue", Arial, sans-serif;`. |
| `has-font-heading` | The font defined in the **heading** field of your theme. By default it will apply `font-family: inherit;`. |
| `has-font-code` | The font defined in the **code** field of your theme. By default it will apply `font-family: SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace;`. |

Use `has-font-body` to apply the same font used by default in your page:

```html live=true
<div class="has-font-body">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit... 
</div>
```

Use `has-font-heading` to apply the same font used by default in your headings:

```html live=true
<div class="has-font-heading">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit... 
</div>
```

Use `has-font-code` to apply the same font that will be applied to the `<code>` and `<pre>` blocks:

```html live=true
<div class="has-font-code">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit... 
</div>
```

### Font size

For each size defined in the `fontSizes` field of your theme there will be a helper with the pattern `has-size-{index}`. In the default theme of **siimple** the following font sizes helpers will be available:

| Class | Property |
|-------|----------|
| `has-size-0`  | `font-size: 0.75rem;` |
| `has-size-1`  | `font-size: 1rem;`    |
| `has-size-2`  | `font-size: 1.25rem;` |
| `has-size-3`  | `font-size: 1.5rem;`  |
| `has-size-4`  | `font-size: 2rem;`    |
| `has-size-5`  | `font-size: 2.5rem;`  |
| `has-size-6`  | `font-size: 3rem;`    |
| `has-size-7`  | `font-size: 3.5rem;`  |
| `has-size-8`  | `font-size: 4rem;`    |
| `has-size-9`  | `font-size: 4.5rem;`  |

Example:

```html live=true
<div class="has-size-0">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-1">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-2">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-3">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-4">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-5">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-6">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-7">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-8">The quick brown fox jumps over the lazy dog.</div>
<div class="has-size-9">The quick brown fox jumps over the lazy dog.</div>
```

### Font Weight

For each weight defined in the `fontWeights` field of your theme there will be a helper with the pattern `has-weight-{name}` to apply the specified weight. The following weight helpers will be always available:

| Class | Property |
|-------|----------|
| `has-weight-body` | Weight used for the body text. In the default theme this helper will apply `font-weight: 400;`. |
| `has-weight-heading` | Weight used for headings. In the default theme this will apply `font-weight: 700;`. |
| `has-weight-bold` | Weight used for bold text. In the default theme this will apply `font-weight: 700;`. |

Example:

```html live=true
<div class="has-weight-body">Text using has-weight-body</div>
<div class="has-weight-heading">Text using has-weight-heading</div>
<div class="has-weight-bold">Test using has-weight-bold</div>
```

### Text alignment

Use one of the following classes to set the horizontal align of your text inside the container element:

| Class | Property |
|-------|----------|
| `has-text-justified` | `text-align: justify;` |
| `has-text-centered`  | `text-align: center;` |
| `has-text-left`      | `text-align: left;` |
| `has-text-right`     | `text-align: right;` |

Use the `has-text-justified` helper to justify the inline contents:

```html live=true
<div class="has-text-justified">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud 
    exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</div>
```

Use the `has-text-left` helper to align inline contents to the left: 

```html live=true
<div class="has-text-left">Left aligned text</div>
```

Use the `has-text-centered` helper to center inline contents:

```html live=true
<div class="has-text-centered">Centered text</div>
```

Use the `has-text-right` helper to align inline contents to the right:

```html live=true
<div class="has-text-right">Right aligned text</div>
```

### Text transform

Use the following helpers to apply some transforms to your text:

| Class | Property |
|-------|----------|
| `is-italic`    | `font-style: italic;` |
| `is-underlined` | `text-decoration: underline;` |
| `is-not-underline` | `text-decoration: none;` |
| `is-uppercase` | `text-transform: uppercase;` |
| `is-lowercase` | `text-transform: lowercase;` |
| `is-capitalized` | `text-transform: capitalize;` |

Use the `is-underlined` to underline the text:

```html live=true
<strong class="is-underlined">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

Use `is-italic` to display text in italic style: 

```html live=true
<strong class="is-italic">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

Use the `is-uppercase` helper to convert all characters of your text to uppercase:

```html live=true
<strong class="is-uppercase">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

Use the `is-lowercase` helper to convert all characters of your text to lowercase:

```html live=true
<strong class="is-lowercase">
    Lorem Ipsum Dolor Sit Amet, Consectetur Adipiscing Elit...
</strong>
```

Use the `is-capitalized` helper to convert the first character of each word of your text to uppercase:

```html live=true
<strong class="is-capitalized">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```
