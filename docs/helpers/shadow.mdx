# Shadow

For each shadow defined in the `shadows` field of your theme there will be a helper with the pattern `has-shadow-{name}` for applying the specified shadow to the element.

The following shadow helpers will be always included, regardless the shadows you provide in your theme configuration:

| Class | Property |
|-------|----------|
| `has-shadow-none`   | `box-shadow: none;` |


In the default theme of **siimple** the following shadow helpers are available:

| Class | Property |
|-------|----------|
| `has-shadow-sm`  | `box-shadow: 0 .25rem 1rem -.125rem rgba(54,63,79,.15),0 0 0 1px rgba(54,63,79,.02);` |
| `has-shadow-md` | `box-shadow: 0 .5rem 1rem -.25rem rgba(54,63,79,.2),0 0 0 1px rgba(54,63,79,.02);` |
| `has-shadow-lg`  | `box-shadow: 0 1rem 1rem -.5rem rgba(54,63,79,.25),0 0 0 1px rgba(54,63,79,.02);` |

Example:

```html live=true centered=true
<div class="has-shadow-sm has-m-2 has-p-6 has-radius-md has-bg-white">
    .has-shadow-sm
</div>
<div class="has-shadow-md has-m-2 has-p-6 has-radius-md has-bg-white">
    .has-shadow-md
</div>
<div class="has-shadow-lg has-m-2 has-p-6 has-radius-md has-bg-white">
    .has-shadow-lg
</div>
```

You can remove the shadow of any element using the `has-shadow-none` helper:

```html live=true centered=true
<div class="has-shadow-none has-m-2 has-p-6 has-radius-md has-bg-white">
    .has-shadow-none
</div>
```
