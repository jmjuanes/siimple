# Sizing

You can use the sizing helpers to set the width or height of any element. 

### Customize sizing helpers

The sizes that will be used to generate the sizing helpers can be defined in the `sizes` field of your theme.

```js title=siimple.config.js
export default {
    sizes: {
        "0": "0px",
        "1": "0.25rem",
        "2": "0.5rem",
        "4": "1rem",
        // ...other sizes
    },
    // ...other configuration
};
``` 

The following sizes will be always included for generating the sizing helpers, regardless the sizes you provide in your theme configuration:

| Name | Value | Notes |
|------|-------|-------|
| `auto` | `auto` | |
| `none` | `0px` | |
| `one`  | `1px` | |
| `half` | `50%` | |
| `full` | `100%` | |
| `screen` | `100vw` | Will be applied only on **width** properties, for example `width` or `max-width`. |
| `screen` | `100vh` | Will be applied only on **height** properties, for example `height` or `max-height`. |

### Width

Add any of the `has-w-{size}` classnames to modify the `width` property of the element.

Use `has-w-auto` to let the browser to automatically calculate the width of the element: 

```html
<div class="has-w-auto ..."></div>
```

Use `has-w-screen` to fit the width of the screen:

```html
<div class="has-w-screen ..."></div>
```

Use the other clasess to set a fixed width:

```html live=true centered=true color=white
<div class="has-w-64 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-w-64</strong>
</div>
```

### Max Width

Use `has-maxw-{size}` classes to set the `max-width` property of any element.

```html live=true centered=true color=white
<div class="has-w-full has-maxw-64 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-maxw-64</strong>
</div>
```

### Min Width

Use `has-minw-{size}` classes to modify the `min-width` property of any element.

```html live=true centered=true color=white
<div class="has-minw-64 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-minw-64</strong>
</div>
```

## Height

Add any of the `has-h-{size}` classnames to modify the `height` property of the element.

Use `has-h-auto` to let the browser to automatically calculate the height of the element: 

```html
<div class="has-h-auto ..."></div>
```

Use `has-h-screen` to fit the height of the screen:

```html
<div class="has-h-screen ..."></div>
```

Use the other clasess to set a fixed height:

```html live=true centered=true color=white
<div class="has-h-32 has-w-64 has-p-4 has-bg-primary has-radius-md">
    <strong>has-h-32</strong>
</div>
```

## Max Height

Use `has-maxh-{size}` classes to modify the `max-height` property of any element.

```html live=true centered=true color=white
<div class="has-h-full has-maxh-64 has-w-64 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-maxh-64</strong>
</div>
```

## Min Height

Use `has-minh-{size}` classes to modify the `min-height` property of any element.

```html live=true centered=true color=white
<div class="has-minh-64 has-w-64 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-minh-64</strong>
</div>
```

### Uniform width and height

Use `has-s-{size}` classes to set both the `width` and `height` properties of any element.

```html live=true centered=true color=white
<div class="has-s-32 has-p-4 has-bg-primary has-radius-md">
    <strong>.has-s-32</strong>
</div>
```

### Responsive sizing

You can control the sizing of an element at a specific breakpoint adding a `-{breakpoint}` suffix to the sizing class.

```html
<div class="has-w-half has-w-full-mobile">
    . . .
</div>
```
