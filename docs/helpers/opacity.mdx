# Opacity

For each opacity value defined in the `opacities` field of your theme there will be a helper with the pattern `has-opacity-{index}` for applying the specified opacity value. In the default theme of **siimple** the following opacities helpers are available:

| Class | Property |
|-------|----------|
| `has-opacity-0` | `opacity: 0.00;` |
| `has-opacity-20` | `opacity: 0.25;` |
| `has-opacity-50` | `opacity: 0.50;` |
| `has-opacity-75` | `opacity: 0.75;` |
| `has-opacity-100` | `opacity: 1.00;` |

Example:

```html live=true color=white
<div class="is-flex has-text-center">
    <div class="has-w-full has-m-2 has-p-4 has-radius-md has-bg-primary has-opacity-0">
        <strong>0%</strong>
    </div>
    <div class="has-w-full has-m-2 has-p-4 has-radius-md has-bg-primary has-opacity-25">
        <strong>25%</strong>
    </div>
    <div class="has-w-full has-m-2 has-p-4 has-radius-md has-bg-primary has-opacity-50">
        <strong>50%</strong>
    </div>
    <div class="has-w-full has-m-2 has-p-4 has-radius-md has-bg-primary has-opacity-75">
        <strong>75%</strong>
    </div>
    <div class="has-w-full has-m-2 has-p-4 has-radius-md has-bg-primary has-opacity-100">
        <strong>100%</strong>
    </div>
</div>
```

You can change the opacity of any element on hover adding the `-hover` suffix to the opacity helper.

```html
<div class="has-opacity-50 has-opacity-100-hover ...">
    . . .
</div>
```

You can also control the opacity of an element at a specific breakpoint adding a `-{breakpoint}` suffix to the opacity class. In the following example, the `has-opacity-100-mobile` will apply `opacity: 1` only on mobile screens.

```html
<div class="has-opacity-50 has-opacity-100-mobile ...">
    . . . 
</div>
```
