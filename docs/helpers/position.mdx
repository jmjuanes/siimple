# Position

#### Bottom

Use the following classes to set the **bottom** position of an element:

| Class | Property |
|-------|----------|
| `has-bottom-auto` | `bottom: auto;` |
| `has-bottom-none` | `bottom: 0px;`  |
| `has-bottom-half` | `bottom: 50%;`  |
| `has-bottom-full` | `bottom: 100%;` |

Example:

```html
<div class="is-fixed has-bottom-none">
    ...
</div>
```

You can control the position of an element at a specific breakpoint adding a `-{breakpoint}` suffix to the class.

```html
<div class="is-fixed-mobile has-bottom-none-mobile">
    . . .
</div>
```
