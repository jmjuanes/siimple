# Overflow

You can use the following helpers to control the behavior for an element's overflow.

| Class | Property |
|-------|----------|
| `has-overflow-auto`     | `overflow: auto;` |
| `has-overflow-scroll`   | `overflow: scroll;` |
| `has-overflow-hidden`   | `overflow: hidden;` |
| `has-overflow-x-auto`   | `overflow-x: auto;` |
| `has-overflow-x-scroll` | `overflow-x: scroll;` |
| `has-overflow-x-hidden` | `overflow-x: hidden;` |
| `has-overflow-y-auto`   | `overflow-y: auto;` |
| `has-overflow-y-scroll` | `overflow-y: scroll;` |
| `has-overflow-y-hidden` | `overflow-y: hidden;` |


### Enable this plugin

This plugin is included as part of the `plugin-helpers` collection. You can manually include this plugin calling `plugin-overflow-helpers()` in the `$plugins` section of the configuration:

```scss
@use "siimple" as siimple;

@include siimple.configure(
    $plugins: (
        siimple.plugin-overflow-helpers(),
    ),
);
```

### Usage

Use the `has-overflow-auto` to display scrollbars only if content does not fit inside the padding box of the container:

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-auto has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```

Use the `has-overflow-scroll` to always display scrollbars in both directions (but keep in mind that some OS will hide scrollbars when not neccesary). Instead of displaying scrollbars in both directions, you can always use `has-overflow-x-scroll` to enable only horizontal scrolling or `has-overflow-y-scroll` to enable only vertical scrolling. 

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-scroll has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```

On the other hand, you can use `has-overflow-hidden` helper to clip all neccesary content to fit the container size. Note that no scrollbars will be displayed and the user scrolling will be disabled.

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-hidden has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```
