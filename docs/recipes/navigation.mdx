# Navigation components

In this recipe you can find some examples for building navigation components using core elements and utilities.

## Tabs component

You can use the [navlink](/elements/navlink) element and some [utilities](/presets/utilities) for building a tabs component for navigation.

```html live=true
<div class="has-d-flex mobile:has-flex-column has-radius has-bg-coolgray-200 has-p-4 has-mb-4">
    <a class="navlink has-text-center">Tab 1</a>
    <a class="navlink has-text-center has-bg-white has-text-blue-500">Tab 2</a>
    <a class="navlink has-text-center">Tab 3</a>
</div>
```

## Pills component

Using the [navlink](/elements/navlink) and similar utilities you can build a pills navigation component:

```html live=true
<div class="has-d-flex mobile:has-flex-column">
    <a class="navlink has-text-center">Pill 1</a>
    <a class="navlink has-text-center has-bg-blue-500 has-text-white">Pill 2</a>
    <a class="navlink has-text-center">Pill 3</a>
</div>
```
