# Tabs

You can use the [navlink](/elements/navlink) element and some [utilities](/presets/utilities) for building a tabs component for navigation.

```html live=true
<div class="has-d-flex mobile:has-flex-column has-radius has-bg-coolgray-200 has-p-2">
    <a class="navlink has-text-center">Tab 1</a>
    <a class="navlink has-text-center has-bg-white has-text-blue-500">Tab 2</a>
    <a class="navlink has-text-center">Tab 3</a>
</div>
```

Also, using the [navlink](/elements/navlink) and similar utilities you can build a pills navigation component instead:

```html live=true
<div class="has-d-flex mobile:has-flex-column">
    <a class="navlink has-text-center">Pill 1</a>
    <a class="navlink has-text-center has-bg-blue-500 has-text-white hover:has-text-white">Pill 2</a>
    <a class="navlink has-text-center">Pill 3</a>
</div>
```
