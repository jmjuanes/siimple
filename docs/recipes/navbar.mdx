# Navbar

<NavbarExample />

```html
<div class="container has-p-6 isflex has-items-center has-flex-wrap">
    <!-- Navigation brand: always visible at the left side of the navbar -->
    <a class="is-block has-mr-auto has-size-2 has-text-current">
        <strong>Navbar</strong>
    </a>
    <!-- Toggler button: only visible in mobile screens -->
    <div id="navbar-toggler" class="menu is-hidden-tablet"></div>
    <!-- Navigation menu: by default hidden on mobile screens -->
    <div id="navbar-menu" class="is-flex-mobile has-w-full-mobile has-direction-column-mobile has-mt-4-mobile is-hidden-mobile">
        <a class="navlink has-mb-2-mobile">Link 1</a>
        <a class="navlink">Link 2</a>
    </div>
</div>
``` 

You can use the following JavaScript implementation example that toggles the visibility of the navigation menu when the toggler button is clicked:

```javascript
const menu = document.querySelector("#navbar-menu");
const toggler = document.querySelector("#navbar-toggler");

toggler.addEventListener("click", () => {
    menu.classList.toggle("is-flex-mobile");
    menu.classList.toggle("is-hidden-mobile");
});
```
