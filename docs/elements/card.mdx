# Card

A **card** is a flexible container element that holds content and actions about a single subject or topic. By default, cards have no fixed width. A card can be created adding a `card` class to a `<div>` tag.


```html live=true
<div class="card has-mx-auto has-maxw-96 has-shadow">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. 
</div>
```

### Customize

The card element will apply the styles defined in the `cards` field of your theme configuration.

```js title=siimple.config.js
export default {
    cards: {
        backgroundColor: "white",
        color: "inherit",
        padding: "2rem",
    },
};
```
