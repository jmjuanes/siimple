# Textarea

Add the `textarea` class to your `<textarea>` element. 

```html live=true
<textarea class="textarea" rows="5">Textarea content</textarea>
```

### Variants

Variants for this element can be defined in the `form.textarea` section of the variants configuration:

```js
export default {
    variants: {
        "form.textarea": {
            default: {
                backgroundColor: "fill",
            },
        },
    },
};
```
