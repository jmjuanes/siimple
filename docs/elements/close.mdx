# Close

Use the `close` class to create a close button:

```html align=center live=true
<div class="close"></div>
```

### Customize

The close element will apply the styles defined in the `buttons.close` field of your theme configuration.

```js title=siimple.config.js
export default {
    buttons: {
        close: {
            color: "currentColor",
        },
    },
};
```
