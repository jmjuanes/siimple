# Close

Use the `close` class to create a close button:

```html align=center live=true
<div class="close"></div>
```


### Variants

Variants for this element can be defined in the `button.close` section of the variants configuration:

```js
export default {
    variants: {
        "button.close": {
            default: {
                color: "currentColor",
            },
        },
    },
};
```
