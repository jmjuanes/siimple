# Title

Use `title` to style your headings. By default we provide six different modifiers to customize the title text size, using the sizes defined in your `fontSizes` scale:

| `fontSize` index | Title modifier |
|------------------|----------------|
| 1 | `is-6` |
| 2 | `is-5` |
| 3 | `is-4` |
| 4 | `is-3` |
| 5 | `is-2` |
| 6 | `is-1` |

Example:

```html live=true
<div class="title is-1">Title 1</div>
<div class="title is-2">Title 2</div>
<div class="title is-3">Title 3</div>
<div class="title is-4">Title 4</div>
<div class="title is-5">Title 5</div>
<div class="title is-6">Title 6</div>
```

### Customize

The title element will apply the styles defined in the `text.heading` field of your theme configuration.

```js title=siimple.config.js
export default {
    text: {
        heading: {
            color: "#000",
            fontWeight: "800",
        },
    },
    // ...other configuration
};
```
