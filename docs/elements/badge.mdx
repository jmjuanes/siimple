# Badge

**Badges** are useful elements for indicating status, attach metadata to another element or displaying numbers for counts. You can easily turn a `<span>` or `<div>` element to a badge adding the `badge` class.

```html live=true
<span class="badge">Badge</span>
<span class="badge is-secondary">Badge</span>
```

We provide an `is-secondary` modifier for using the secondary color defined in your theme as the background color of the badge:

```html live=true
<span class="badge is-secondary">Badge</span>
```

### Variants

Variants for this element can be defined in the `badges` section of your theme configuration. Badges will use the properties defined in `badges.default` to customize the default badge class.

```js
export default {
    badges: {
        // Will be applied to the base '.badge' class
        default: {
            backgroundColor: "primary",
            color: "white",
        },
    },
};
```

Additional fields defined in the `badges` section will generate additional modifierts for the badge element.

```js
export default {
    badges: {
        // Will generate '.badge.is-secondary'
        secondary: {
            backgroundColor: "secondary",
            color: "white",
        },
    },
};
```
