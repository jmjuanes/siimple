# Buttons

Buttons are essential elements in any website or application for **actions**. The `button` class can be used in a `<button>`, `<input type="submit">`, `<a>` or `<div>` tag.

```html live=true
<button class="button">Primary button</button>
```

You can style your button using the secondary color of your theme with the mofidier `is-secondary`:

```html live=true
<button class="button is-secondary">Secondary button</button>
```

We also provide the `is-full` modifier to change the button width to the **full parent width**:

```html live=true
<button class="button is-full">
    Ful width button
</button>
```

### Variants

Variants for this element can be defined in the `buttons` section of your theme configuration. Buttons will use the properties defined in `buttons.default` to customize the default button class.

```js
export default {
    buttons: {
        // Will be applied to the base '.buttons' class
        default: {
            backgroundColor: "primary",
            color: "white",
        },
    },
};
```

Additional fields defined in the `buttons` section will generate additional modifierts for the button element.

```js
export default {
    buttons: {
        // Will generate '.button.is-secondary' 
        secondary: {
            backgroundColor: "secondary",
            color: "white",
        },
        // Will generate '.button.is-disabled'
        disabled: {
            cursor: ["not-allowed", "!important"],
            opacity: ["0.6", "!important"],
        },
        // Will generate '.button.is-full'
        full: {
            display: "block",
            width: "100%",
        },
    },
};
```
