# Buttons

Buttons are essential elements in any website or application for **actions**. The `button` class can be used in a `<button>`, `<input type="submit">`, `<a>` or `<div>` tag.

```html live=true
<button class="button">Primary button</button>
```

You can style your button using the secondary color of your theme with the mofidier `is-secondary`:

```html live=true
<button class="button is-secondary">Secondary button</button>
```

We also provide the `is-full` modifier to change the button width to the **full parent width**:

```html live=true
<button class="button is-full">
    Ful width button
</button>
```

### Customize

The button element will apply the styles defined in the `buttons.default` field of your theme configuration.

```js title=siimple.config.js
export default {
    buttons: {
        default: {
            backgroundColor: "primary",
            color: "white",
            // Will generate '.button.is-disabled' modifier
            "&.is-disabled": {
                cursor: ["not-allowed", "!important"],
                opacity: ["0.6", "!important"],
            },
        },
    },
};
```
