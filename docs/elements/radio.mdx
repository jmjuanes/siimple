# Radio

Add a `radio` class to an `<input type="radio">` element to style it as a radio. 

```html live=true
<label class="label" for="radio0">
    <input type="radio" id="radio0" class="radio has-mr-3">
    Default radio
</label>
```

By default, the radio will be displayed unchecked, but you can add a `checked` attribute to the `<input>` element for displaying the radio as checked.

```html live=true
<label class="label" for="radio1">
    <input type="radio" id="radio1" class="radio has-mr-2" checked >
    Active radio
</label>
```

### Customize

The radio element will apply the styles defined in the `forms.radio` field of your theme configuration.

```js title=siimple.config.js
export default {
    forms: {
        radio: {
            color: "primary",
        },
    },
    // ...other configuration
};
```
