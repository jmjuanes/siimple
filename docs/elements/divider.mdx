# Divider

The **divider** element helps you to separate content in your page. Add `divider` class to a `<div>` or a `<span>` tag to style it as a horizontal rule.

```html live=true
<div class="divider"></div>
```

### Variants

Variants for this element can be defined in the `layout.divider` section of your theme configuration:

```js
export default {
    layout: {
        divider: {
            backgroundColor: "muted",
            height: "0.125rem",
        },
    },
};
```
