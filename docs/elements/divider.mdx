# Divider

The **divider** element helps you to separate content in your page. Add `divider` class to a `<div>` or a `<span>` tag to style it as a horizontal rule.

```html live=true
<div class="divider"></div>
```

### Customize

The divider element will apply the styles defined in the `styles.hr` field of your theme configuration.

```js title=siimple.config.js
export default {
    styles: {
        hr: {
            backgroundColor: "muted",
            height: "0.125rem",
        },
    },
    // ...other configuration
};
```
