# Menu

Use the `menu` class to create a menu button, useful for displaying hidden menus:

```html align=center live=true
<button class="menu"></button>
```


### Variants

Variants for this element can be defined in the `dialogs.menu` field of the variants configuration:

```js
export default {
    variants: {
        dialogs: {
            menu: {
                borderColor: "currentColor",
                color: "currentColor",
            },
        },
    },
};
```
