# Menu

Use the `menu` class to create a menu button, useful for displaying hidden menus:

```html align=center live=true
<button class="menu"></button>
```


### Variants

The default variant for this element can be defined in the `dialog.menu` field of your theme configuration:

```js
export default {
    dialog: {
        menu: {
            borderColor: "currentColor",
            color: "currentColor",
        },
    },
};
```
