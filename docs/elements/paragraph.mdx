# Paragraph

Style your paragraph tag by adding the `paragraph` class:

```html live=true
<p class="paragraph">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</p>
```

We provide a `lead` variant class to highlight a paragraph:

```html live=true
<p class="paragraph is-lead">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</p>
```

### Variants

Variants for this element can be defined in the `text.paragraph` section of the variants configuration:

```js
export default {
    variants: {
        "text.paragraph": {
            default: {
                color: "text",
                fontWeight: "body",
                lineHeight: "body",
            },
            lead: {
                color: "muted",
                fontSize: "large",
                fontWeight: "bold",
            },
        },
    },
};
```
