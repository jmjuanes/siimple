# Paragraph

Style your paragraph tag by adding the `paragraph` class:

```html live=true
<p class="paragraph">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud 
    exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</p>
```

We provide a `is-lead` modifier class to highlight a paragraph:

```html live=true
<p class="paragraph is-lead">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud 
    exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</p>
```

### Customize

The paragraph element will apply the styles defined in the `text.paragraph` field of your theme configuration.

```js title=siimple.config.js
export default {
    text: {
        paragraph: {
            color: "text",
            fontWeight: "body",
            lineHeight: "body",
            "&.is-lead": {
                fontSize: "2",
                fontWeight: "bold",
            },
        },
    },
    // ...other configuration
};
```
