# Navbar

The navbar component allows you to create a responsive navigation header. This component provides support for two additional components:

- `navbrand`: useful for displaying your branding text or logo.
- `navmenu`: useful for displaying navigation links and elements that will be hidden by default in mobile screens.

You can combine the navbar component with the [toggler button](/components/toggler) component, for displaying the navigation menu on mobile screens when the user clicks on the toggler button.

For displaying the navigation menu, we provide an `active` variant that can be added using JavaScript when the user clicks on the toggler button for displaying or hidding the navigation menu. 

<NavbarExample />

```html
<div class="navbar has-radius">
    <a class="navbrand">Navbar</a>
    <!-- The toggler button will be displayed only on mobile screens -->
    <div class="toggler tablet:has-d-none"></div>
    <!-- The navigation menu will be hiddien by default in mobile screens -->
    <!-- unless you add the 'is-active' variant class -->
    <div class="navmenu">
        <a class="navlink">Link 1</a>
        <a class="navlink">Link 2</a>
    </div>
</div>
<!-- Very basic example implementation for displaying or hidding the menu -->
<script type="text/javascript">
    document.querySelector(".toggler").addEventListener("click", () => {
        document.querySelector(".navmenu").classList.toggle("is-active");
    });
</script>
```

> **Tip**: resize your screen to test how navbar works in mobile screens.

> Remember that **siimple does not come with any JavaScript**. However, you can use the implementation example that toggles the variant class `is-active` in the navigation menu component when the toggler button is clicked.


### Variants

Variants for this component can be defined in the `navigation.navbar` section of the variants configuration:

```scss
@include siimple.configure(
    $variants: (
        "navigation.navbar": (
            "default": (
                "background-color": white,
                "color": currentColor,
            ),
        ),
    ),
);
```

Variants for the navigation brand can be defined in the `navigation.navbar.brand` section, and for the navigation menu can be defined in the `navigation.navbar.menu` section.
