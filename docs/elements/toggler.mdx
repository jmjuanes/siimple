# Toggler

Use the `toggler` class to create a toggle button, useful for displaying hidden menus:

```html align=center live=true
<button class="toggler"></button>
```


### Variants

Variants for this element can be defined in the `button.toggler` section of the variants configuration:

```js
export default {
    variants: {
        "button.toggler": {
            default: {
                borderColor: "currentColor",
                color: "currentColor",
            },
        },
    },
};
```
