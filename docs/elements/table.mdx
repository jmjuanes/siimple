# Table

The `table` class allows you to style your `<table>` elements. We provide some modifiers to the default table style:

- `is-divided`: display a divider rule between table rows.
- `is-bordered`: display a table with borders.
- `is-hoverable`: highlight the row where the cursor is.
- `is-fixed`: automatically divide equally the width of the columns across the table, regardless of content inside the cells.

| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |

```html
<table class="table is-divided">
    ...
</table>
```

You can also use the special `table-responsive` class to allow creating responsive tables.

```html
<div class="table-responsive">
    <table class="table">
        . . .
    </table>
</div>
```

### Customize

The table element will apply the styles defined in the `styles.table` field of your theme configuration.

```js title=siimple.config.js
export default {
    styles: {
        table: {
            "& th, & td": {
                backgroundColor: "white",
                padding: "1rem",
            },
        },
    },
    // ...other configuration
};
```
