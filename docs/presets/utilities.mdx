# Utilities

The **utilities** plugin provides a collection of **small functional CSS classes** that can be used for creating or customizing components. The main aim of this module is to provide a collection of reusable classes to replace the conventional style of writting CSS classes.

### Disable utilities

The utilities module are enabled by default, but can be disabled setting to `false` the `useUtilities` flag in your configuration:

```js title=siimple.config.js
export default {
    useUtilities: false, // Skip utilities
};
```
