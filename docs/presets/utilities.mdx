# Utilities preset

The **utilities** preset provides a collection of **small functional styles** that can be used for creating or customizing elements. The main aim is to provide a collection of reusable classes to replace the conventional style of writting CSS classes.

```html
<button class="has-p-2 has-radius has-bg-blue-500 has-text-white">
    <strong>My button</strong>
</button>
```

## Adding utilities preset

This preset can be installed using **npm**:

```bash
$ npm install --save @siimple/preset-utilities
```

In your configuration file, include the preset styles:

```js title=siimple.config.js
import utilities from "@siimple/preset-utilities";

export default {
    // ...other configuration
    styles: {
        ...utilities.styles,
        // ...other styles
    },
};
```

## Using utilities

Utilities defined in this preset follows the pattern `.has-<shortcut>-<value>`.

```html
<div class="has-opacity-50 has-s-64 has-bg-blue-900">
    ...
</div>
```

The following utilities are included as part of this preset:

- [Background](#background) utilities, to control the background of any element.
- [Cursor](#cursor) utilities for changing the cursor style.
- [Display](#display) utilities, to control how elements will be displayed..
- [Flexbox](#flexbox) utilities, that include flex direction, order and more!
- [Opacity](#opacity) utilities, to control the opacity of an element.
- [Overflow](#overflow) utilities, to control the overflow behavior of any element.
- [Placement](#placement) utilities, to set the placement of any element.
- [Radius](#radius) utilities, to change the border radius of an element.
- [Shadow](#shadow) utilities, for adding or removing shadow effects.
- [Sizing](#sizing) utilities, to control the size of an element.
- [Spacing](#spacing) utilities, to control the margin and padding of an element.
- [Typography](#typography) utilities, that include fonts, text align, text color and more!.

### Background

Use `has-bg-{color}` to set the background color of any element. 

```html
<div class="button has-bg-royal-600 has-text-white">
    <strong>Button</strong>
</div>
```

You can change the background color of any element on hover adding the `hover:` prefix to the background color utility.

```html
<div class="button hover:has-bg-green-600 has-bg-aqua-600 has-text-white">
    <strong>Hover me</strong>
</div>
```

### Cursor

Use the following classes to control the cursor property of any element:

| Class | Property |
|-------|----------|
| `has-cursor-pointer`     | `cursor: pointer;` |
| `has-cursor-move`        | `cursor: move;` |
| `has-cursor-none`        | `cursor: none;` |
| `has-cursor-zoom-in`     | `cursor: zoom-in;` |
| `has-cursor-zoom-out`    | `cursor: zoom-out;` |
| `has-cursor-not-allowed` | `cursor: not-allowed;` |
| `has-cursor-wait`        | `cursor: wait;` |

Use `has-cursor-hand` for displaying a pointer cursor to indicate a link or an interactive element:

```html live=true centered=true bg=blue-200 color=white
<div class="has-bg-blue-500 has-text-center has-p-3 has-w-64 has-cursor-pointer">
    Pointer
</div>
```

Use `has-cursor-move` to indicate that the element can be moved:

```html live=true centered=true bg=blue-200 color=white
<div class="has-bg-blue-500 has-text-center has-p-3 has-w-64 has-cursor-move">
    Move
</div>
```

Use `has-cursor-none` to hide the mouse cursor:

```html live=true centered=true bg=blue-200 color=white
<div class="has-bg-blue-500 has-text-center has-p-3 has-w-64 has-cursor-none">
    Cursor disabled
</div>
```

Use `has-cursor-zoom-in` and `has-cursor-zoom-out` for displaying zooming cursors:

```html live=true centered=true bg=blue-200 color=white
<div class="has-bg-blue-500 has-text-center has-p-3 has-mx-2 has-w-32 has-cursor-zoom-in">
    Zoom in
</div>
<div class="has-bg-blue-500 has-text-center has-p-3 has-mx-2 has-w-32 has-cursor-zoom-out">
    Zoom out
</div>
```

### Display

Control how elements will be displayed using the following classes: 

| Class | Property | 
|-------|----------|
| `has-d-none`         | `display: none;` |
| `has-d-inline`       | `display: inline;` |
| `has-d-block`        | `display: block;` |
| `has-d-inline-block` | `display: inline-block;` |
| `has-d-flex`         | `display: flex;` |
| `has-d-inline-flex`  | `display: inline-flex;` |


Use `has-d-none` to hide an element:

```html live=true centered=true bg=blue-200 color=white
<div class="has-bg-blue-500 has-p-3 has-mx-2 has-w-24 has-radius">1</div>
<div class="has-bg-blue-500 has-p-3 has-mx-2 has-w-24 has-radius has-d-none">2</div>
<div class="has-bg-blue-500 has-p-3 has-mx-2 has-w-24 has-radius">3</div>
```

Use `has-d-block` to generate a block-level element: 

```html live=true bg=blue-200 color=white
<span class="has-bg-blue-500 has-p-3 has-m-2 has-radius has-d-block">1</span>
<span class="has-bg-blue-500 has-p-3 has-m-2 has-radius has-d-block">2</span>
<span class="has-bg-blue-500 has-p-3 has-m-2 has-radius has-d-block">3</span>
```

Use `has-d-inline-block` to generate an inline block element: 

```html live=true bg=blue-200 color=primary
<div class="has-bg-blue-500 has-radius has-p-2">
    <div class="has-bg-white has-p-3 has-m-2 has-radius has-d-inline-block">1</div>
    <div class="has-bg-white has-p-3 has-m-2 has-radius has-d-inline-block">2</div>
    <div class="has-bg-white has-p-3 has-m-2 has-radius has-d-inline-block">3</div>
</div>
```

You can control the display of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the display class. For example, to hide an element only on mobile screens, use the class `mobile:has-display-none`.

```html
<div class="has-d-block mobile:has-d-none">
    . . . 
</div>
```

### Flexbox

#### Flex

Use the following classes to control the behaviour of flex items:

| Class | Property |
|-------|----------|
| `has-flex-auto`    | `flex: auto;`    |
| `has-flex-initial` | `flex: initial;` |
| `has-flex-none`    | `flex: none;`    |

You can use `has-flex-auto` helper to allow the item to grow and shrink to fit the container:

```html live=true bg=blue-200 color=white
<div class="has-d-flex">
    <div class="has-flex-auto has-m-2 has-p-4 has-radius has-bg-blue-500">
        Small
    </div>
    <div class="has-flex-auto has-m-2 has-p-4 has-radius has-bg-blue-500">
        Medium content
    </div>
    <div class="has-flex-auto has-m-2 has-p-4 has-radius has-bg-blue-500">
        Large content for this flexbox item
    </div>
</div>
```

You can use the `has-flex-initial` helper to allow the item to shrink, but prevent the item to grow to fit the container.

```html live=true bg=blue-200 color=white
<div class="has-d-flex">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500 has-w-full">
        Item
    </div>
    <div class="has-flex-initial has-m-2 has-p-4 has-radius has-bg-blue-700">
        Lorem ipsum dolor sit amet, consectetur adipisicing elit. 
        Qui ad labore ipsam, aut rem quo repellat esse tempore id, quidem.
    </div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500 has-w-full">
        Item
    </div>
</div>
```

Use the `has-flex-none` helper to prevent a flex item from growing and shrinking:

```html live=true bg=blue-200 color=white
<div class="has-d-flex">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500 has-w-full">
        Item
    </div>
    <div class="has-flex-none has-m-2 has-p-4 has-radius has-bg-blue-700">
        This can not grow or shrink
    </div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500 has-w-full">
        Item
    </div>
</div>
```

#### Flex direction

Use the following classes to control the direction of the items of a flex container:

| Class | Property |
|-------|----------|
| `has-flex-row`        | `flex-direction: row;`           |
| `has-flex-row-rev`    | `flex-direction: row-reverse`    |
| `has-flex-column`     | `flex-direction: column;`        |
| `has-flex-column-rev` | `flex-direction: column-reverse` |

Use `has-flex-row` to position flex items horizontailly (row direction):

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-flex-row">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">2</div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

Or just use `has-flex-column` to position flex items vertically (in a column direction):

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-flex-column">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">2</div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

#### Align items

Use the following classes to define the alignment of the flex items along the **cross axis**.

| Class | Property |
|-------|----------|
| `has-items-start`    | `align-items: flex-start;` |
| `has-items-end`      | `align-items: flex-end;`   |
| `has-items-center`   | `align-items: center;`     |
| `has-items-stretch`  | `align-items: stretch;`    |

You can use `has-items-start` to align flex items to the start of the cross axis:

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-items-start has-text-center has-h-32">
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-flex-grow has-m-2 has-p-8 has-radius has-bg-blue-500">2</div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

Use `has-items-center` to align flex items to the center of the cross axis:

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-items-center has-text-center has-h-32">
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-flex-grow has-m-2 has-p-8 has-radius has-bg-blue-500">2</div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

Use `has-items-end` helper to align flex items to the end of the cross axis:

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-items-end has-text-center has-h-32">
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-flex-grow has-m-2 has-p-8 has-radius has-bg-blue-500">2</div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

| Class | Property |
|-------|----------|
| `has-justify-left`    | `justify-content: flex-start;`    |
| `has-justify-center`  | `justify-content: center;`        |
| `has-justify-right`   | `justify-content: flex-end;`      |
| `has-justify-between` | `justify-content: space-between;` |
| `has-justify-around`  | `justify-content: space-around;`  |
| `has-justify-evenly`  | `justify-content: space-evenly;`  |

Use the `has-justify-start` helper to pack items from the start of the main axis:

```html live=true bg=blue-200 color=white

    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>
</div>
```

Use the `has-justify-center` helper to pack items around the center of the main axis:

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-justify-center has-text-center">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">2</div>
```

Use the `has-justify-end` helper to pack items from the end of the main axis:

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-justify-end has-text-center">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>

    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">3</div>

```

Use the `has-justify-between` helper to distribute items evenly, and where the first item is flush with the start and the last item is flush with the end.

```html live=true bg=blue-300 color=white
<div class="has-d-flex has-justify-between has-text-center">
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
```

Use the `has-justify-around` helper to distribute items evenly, and items will have a half-size space on either end:

```html live=true bg=blue-200 color=white
    <div class="has-m-2 has-p-4 has-radius has-bg-blue-500">1</div>
```

### Opacity

Add any of the `has-opacity-{value}` classes to set the opacity of an element. We are using [this opacities](https://github.com/jmjuanes/siimple/blob/main/siimple/theme.js#L74-L96) values.

```html live=true bg=blue-200 color=white
<div class="has-d-flex has-text-center">
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500 has-opacity-0">
        <strong>0%</strong>
    </div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500 has-opacity-25">
        <strong>25%</strong>
    </div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500 has-opacity-50">
        <strong>50%</strong>
    </div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500 has-opacity-75">
        <strong>75%</strong>
    </div>
    <div class="has-flex-grow has-m-2 has-p-4 has-radius has-bg-blue-500 has-opacity-100">
        <strong>100%</strong>
    </div>
</div>
```

You can change the opacity of any element on hover adding the `hover:` prefix to the opacity helper.

```html
<div class="has-opacity-50 hover:has-opacity-100 ...">
    . . .
</div>
```

You can control the opacity of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the opacity class.

```html
<div class="has-opacity-50 mobile:has-opacity-100 ...">
    . . . 
</div>
```

### Overflow

You can use the following utilities to control the behavior for an element's overflow.

| Class | Property |
|-------|----------|
| `has-overflow-auto`     | `overflow: auto;` |
| `has-overflow-scroll`   | `overflow: scroll;` |
| `has-overflow-hidden`   | `overflow: hidden;` |
| `has-overflow-x-auto`   | `overflow-x: auto;` |
| `has-overflow-x-scroll` | `overflow-x: scroll;` |
| `has-overflow-x-hidden` | `overflow-x: hidden;` |
| `has-overflow-y-auto`   | `overflow-y: auto;` |
| `has-overflow-y-scroll` | `overflow-y: scroll;` |
| `has-overflow-y-hidden` | `overflow-y: hidden;` |

Use `has-overflow-auto` to display scrollbars only if content does not fit inside the padding box of the container:

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-auto has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```

Use `has-overflow-scroll` to always display scrollbars in both directions (but keep in mind that some OS will hide scrollbars when not neccesary). Instead of displaying scrollbars in both directions, you can always use `has-overflow-x-scroll` to enable only horizontal scrolling or `has-overflow-y-scroll` to enable only vertical scrolling. 

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-scroll has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```

On the other hand, you can use `has-overflow-hidden` to clip all neccesary content to fit the container size. Note that no scrollbars will be displayed and the user scrolling will be disabled.

```html live=true color=white bg=blue-200 centered=true
<div class="has-overflow-hidden has-h-32 has-maxw-64 has-p-6 has-bg-blue-500 has-radius">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. 
    Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut 
    aliquip ex ea commodo consequat.
</div>
```

### Placement
#### Position 

Use the following classes to change the position of an element:

| Class | Property | 
|-------|----------|
| `has-position-initial`  | `position: initial;` |
| `has-position-relative` | `position: relative;` |
| `has-position-absolute` | `position: absolute;` |
| `has-position-sticky`   | `position: sticky;` |
| `has-position-fixed`    | `position: fixed;` |

Example:

```html
<div class="has-position-fixed has-top-none">
    ...
</div>
```

You can control the position of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the class.

```html
<div class="mobile:has-position-fixed mobile:has-top-none">
    . . .
</div>
```

### Spacing

For all spacing utilities the following [sizes](https://github.com/jmjuanes/siimple/blob/main/siimple/theme.js#L54-L70) can be used.

#### Uniform padding or margin

Use `has-p-{size}` to set the **padding** of all sides of the element, or `has-m-{size}` to set the **margin** of all sides of the element.

```html live=true centered=true bg=blue-200 color=white
<div class="has-p-6 has-bg-blue-500 has-radius">
    <strong>has-p-6</strong>
</div>
```

You can control the spacing of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the padding or margin class.

```html
<div class="has-p-3 mobile:has-p-0">
    . . .
</div>
``` 

#### Directional padding or margin

Use `has-p{direction}-{size}` or `has-m{direction}-{size}` to set the **padding or margin of one side**, **on the x-axis** or **the y-axis** of the element. The following sides are available:

- `t`: to set the margin or padding **top**. Example: `has-pt-none`.
- `b`: to set the margin or padding **bottom**. Example: `has-pb-none`.
- `l`: to set the margin or padding **left**. Example: `has-pl-none`.
- `r`: to set the margin or padding **right**. Example: `has-pr-none`.
- `x`: to set the margin or padding **left and right**. Example: `has-px-none`.
- `y`: to set the margin or padding **top and bottom**. Example: `has-py-none`.

```html live=true centered=true bg=blue-200 color=white padding=12
<div class="has-p-4 has-pt-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pt-12</strong>
</div>
<div class="has-p-4 has-pr-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pr-12</strong>
</div>
<div class="has-p-4 has-pb-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pb-12</strong>
</div>
<div class="has-p-4 has-pl-12 has-m-4 has-bg-blue-500 has-radius">
    <strong>has-pl-12</strong>
</div>
```

We provide an additional `has-mx-auto` classname to set the margin left and right to `auto`. This will center the block with a fixed width.

```html live=true bg=blue-200 color=white
<div class="has-mx-auto has-p-3 has-w-32 has-bg-blue-500 has-radius">
    <strong>Centered</strong>
</div>
```

You can control the spacing of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the padding or margin class.

```html
<div class="has-py-3 mobile:has-py-0">
    . . .
</div>
```
