# Icons

We provide a collection of hand-crafted and pure CSS icons that can be used as a preset for the **siimple** toolkit. After enabling the preset, you can use the classname pattern `icon-<icon-name>` for including the icons in your HTML:

```html
<i class="icon-star"></i>
```

### Adding icons

The icons package is included when you install the `siimple` package framework, so you can just import icons styles in your `siimple.config.js` file:

```js title=siimple.config.js
import icons from "@siimple/preset-icons";

export default {
    styles: {
        ...icons.styles,
        // ...other styles
    },
    // ...other configuration
};
```
