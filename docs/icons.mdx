# Icons

We provide a collection of hand-crafted and open source icons webfont that can be used in any web and mobile project.

### Enable icons plugin

Icons are enabled by default, but can be disabled setting to `false` the `icons` field in the `corePlugins` section of the configuration.

```js title=siimple.config.js
export default {
    corePlugins: {
        icons: false, // Skip icons
    },
};
```

### Using icons

Create a `<i>` tag and add the `icon-<IconName>` classname, where `<IconName>` is the name of the desired icon.

```html 
<!-- Add a left arrow icon -->
<i class="icon-arrow-left"></i>

<!-- Add a star icon -->
<i class="icon-star"></i>
```

You can style your icon using `font-size` and `color` style properties:

```html 
<!-- Using style attribute -->
<i class="icon-star" style="color:#ffea66;font-size:20px;"></i>

<!-- Using helpers -->
<i class="icon-star has-color-red-500 has-size-xl"></i>
```
