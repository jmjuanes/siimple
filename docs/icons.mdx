# Icons


### Icons cheatsheet


### Enable icons module

The icons module is by default enabled as part of the `siimple.css()` mixin. It can be disabled setting a `false` value to the `"use-icons"` flag in the `$flags` section of the configuration.

Also, it can be manually included in your styles just calling the `siimple.icons()` mixin:

```scss
@use "siimple" as siimple;

// Include icons styles
@include siimple.icons();
```

### Customize

The following options can be used to customize the icons module:

- `"font-path"`: path where the icons fonts are placed. By default is `./`. **The path must end with a `/`**.

Example: 

```scss
@use "siimple" as siimple;

// Include and configure icons
@include siimple.icons((
    "font-path": "./fonts/",
));
```

> Remember to copy all fonts in the `siimple` package to your project assets folder and use the `"font-path"` option to set the path to the fonts folder.


### Using icons

Create a `<i>` tag and add the `icon-<IconName>` classname, where `<IconName>` is the name of the desired icon.

```html 
<!-- Add a left arrow icon -->
<i class="icon-arrow-left"></i>

<!-- Add a star icon -->
<i class="icon-star"></i>
```

You can style your icon using `font-size` and `color` style properties:

```html 
<!-- Using style attribute -->
<i class="icon-star" style="color:#ffea66;font-size:20px;"></i>

<!-- Using helpers -->
<i class="icon-star has-color-red-500 has-size-xl"></i>
```
