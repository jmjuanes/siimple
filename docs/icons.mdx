# Icons

We provide a collection of hand-crafted and pure CSS icons that can be used as a preset for the **siimple** toolkit. After enabling the preset, you can use the classname pattern `icon-<icon-name>` for including the icons in your HTML:

```html
<i class="icon-star"></i>
```
