# @siimple/preset-ice

> A frosty and clean theme for **siimple**.


## Installation

Install this package using **npm**:

```
$ npm install --save siimple @siimple/preset-ice
```

## Usage

Use this theme in your `siimple.config.js` file:

```js
import ice from "@siimple/preset-ice";

export default {
    ...ice,
    // ...other configuration
};
```
