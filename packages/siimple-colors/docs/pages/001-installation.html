Get started with siimple colors adding it in your proyect. There are three ways to install it:
Using a package manager
You can easily download and install in your project the latest version of siimple colors using npm or yarn: 
$ npm install --save @siimple/colors
$ yarn add @siimple/colors
Now you can include the CSS of siimple colors in your project adding the following line in the <head> of your HTML document: 
<link rel="stylesheet" href="./node_modules/@siimple/colors/dist/siimple-colors.min.css">
Using a CDN
Import all CSS styles and classes of siimple colors adding the following line in the <head> of each HTML document of your project or website: 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@siimple/colors@{{ site.version }}/dist/siimple-colors.min.css">
